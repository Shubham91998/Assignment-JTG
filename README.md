# Homepage Website

This is a simple Homepage website assignment made using only HTML, CSS, and JavaScript.

The project is created for Josh Technology Group home assignment.

## Features

- Responsive Navbar
- Hero Section
- Projects Section
- Recommendations Slider
- Contact Form
- Footer Section
- Responsive Design for Mobile and Tablet
- Active Navbar Effect using JavaScript
- Recommendation Slider using JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

No library or framework is used in this project.

## JavaScript Functionalities

### 1. Navbar Active Effect
When user clicks on navbar items, active color changes automatically.

### 2. Recommendation Slider
User can click on dots to move recommendation cards.

## Responsive Design

Media queries are used for:
- Tablet devices
- Mobile devices

Website adjusts properly on different screen sizes.

## How to Run Project

1. Download or clone the project
2. Open project folder
3. Run `index.html` in browser
